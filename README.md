# Generative Surfboards
Generates random vector surfboard shapes & designs, intended for pen plotting.

<p style="text-align:center">
  <img src="https://github.com/davidhampgonsalves/generative-surfboards/blob/master/sample.png" width="100%">
</p>

## Setup
```
git clone git@github.com:davidhampgonsalves/generative-surfboards.git
cd generative-surfboards
yarn install
node index.js
// output surfboard svg will be in the out dir
```

### Run Multiple Times (zsh)
```
repeat 50 {node index.js}
```
